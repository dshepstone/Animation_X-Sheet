========================================================================
           ANIMATION X-SHEET  v1.0.0   –   OFFLINE BUILD
           Zero-install timing-sheet tool for animators
========================================================================

1.  QUICK START
   ---------------------------------------------------------------------
     1) UNZIP  this file anywhere you like (Desktop, USB key, etc.).
     2) OPEN   the new “Animation_X-Sheet” folder and double-click
        index.html   (or right-click → “Open with” → your browser).
     3) START  blocking, sketching thumbnails and exporting PDFs.
        – Works entirely offline; no installation or admin rights.

2.  FOLDER CONTENTS
   ---------------------------------------------------------------------
     Animation_X-Sheet/
       ├─ index.html          main application
       ├─ css/                interface styling
       ├─ js/                 JavaScript modules
       └─ assets/             icons / images (optional)

     Feel free to move or copy the folder; everything is self-contained.

3.  SYSTEM REQUIREMENTS
   ---------------------------------------------------------------------
     • Browser :  Chrome 92+, Edge 92+, Firefox 90+, Safari 15+
     • OS      :  Windows 10+, macOS 10.15+, Linux (Ubuntu 18.04+),
                  Chrome OS, iPadOS 15+
     • Hardware :  4 GB RAM, 100 MB free disk, mouse/trackpad or tablet

4.  SAVING & LOADING
   ---------------------------------------------------------------------
     ● Save project   :  File → Save Project…  (creates .axs.json file)
     ● Load project   :  File → Load Project…  (select a .axs.json file)
     ● Export to PDF  :  File → Export PDF…    (auto-paginates sheets)

     All files stay local; nothing is uploaded.

5.  GETTING HELP
   ---------------------------------------------------------------------
     • How-To Guide   : press F1 inside the app
     • GitHub Issues  : https://github.com/dshepstone/Animation_X-Sheet/issues
     • Community      : https://github.com/dshepstone/Animation_X-Sheet/discussions

7.  UPDATING TO A NEW VERSION
   ---------------------------------------------------------------------
     1) Download the latest zip from
        https://github.com/dshepstone/Animation_X-Sheet/releases/latest
     2) Unzip and overwrite the old folder (or keep versions side-by-side).
     3) Your .axs.json project files remain compatible across updates.
